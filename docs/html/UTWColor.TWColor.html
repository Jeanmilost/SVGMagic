IEHexFormat = (...);

Hexadecimal format enumeration

Values
  • IE_HF_RGB: Hexadecimal will be formatted as Red, Green and Blue value (#RRGGBB)
  • IE_HF_BGR: Hexadecimal will be formatted as Blue, Green and Red value (#BBGGRR)
  • IE_HF_ARGB: Hexadecimal will be formatted as Alpha, Red, Green and Blue value (#AARRGGBB)
  • IE_HF_RGBA: Hexadecimal will be formatted as Red, Green, Blue and Alpha value (#RRGGBBAA)
  • IE_HF_ABGR: Hexadecimal will be formatted as Alpha, Blue, Green and Red value (#AABBGGRR)
  • IE_HF_BGRA: Hexadecimal will be formatted as Blue, Green, Red and Alpha value (#BBGGRRAA)
