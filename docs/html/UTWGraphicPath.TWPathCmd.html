type TWPathCmd = class(TObject)

Description

Path command (or instruction). Each of them notify the converter about the next action to do, for example go to position [x, y] or draw a line from current position to [x, y]

IPoints = array of Single;

List of points making up the command, his content depends of the command itself. For example, a IE_IT_MoveTo command will contain 2 points, where points[0] is the x coordinate and points[1] the y coordinate to go to
