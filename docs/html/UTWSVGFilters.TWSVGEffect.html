Nested Types

IEType = (...);

Filter effect type enumeration

Values
  • IE_ET_Unknown: Unknown effect
  • IE_ET_Blend: Filter for compose two objects together ruled by a certain blending mode
  • IE_ET_ColorMatrix: Filter for change colors based on a transformation matrix
  • IE_ET_ComponentTransfer: Filter for perform color-component-wise remapping of data for each pixel
  • IE_ET_Composite: Filter for perform the combination of two input images pixel-wise in image space using one of the Porter-Duff compositing operations
  • IE_ET_ConvolveMatrix: Filter for apply a matrix convolution filter effect
  • IE_ET_DiffuseLighting: Filter for light an image using the alpha channel as a bump map
  • IE_ET_DisplacementMap: Filter for use the pixel values from the image from in2 to spatially displace the image from in
  • IE_ET_Flood: Filter for fill the filter subregion with the color and opacity defined by flood-color and flood-opacity
  • IE_ET_GaussianBlur: Filter for blur the input image by the amount specified in stdDeviation, which defines the bell-curve
  • IE_ET_Image: Filter for fetch image data from an external source and provides the pixel data as output (meaning if the external source is an SVG image, it is rasterized)
  • IE_ET_Merge: Filter for allow filter effects to be applied concurrently instead of sequentially
  • IE_ET_Morphology: Filter for erode or dilate the input image
  • IE_ET_Offset: Filter for offset the input image
  • IE_ET_SpecularLighting: Filter for light a source graphic using the alpha channel as a bump map
  • IE_ET_Tile: Filter to fill a target rectangle with a repeated, tiled pattern of an input image
  • IE_ET_Turbulence: Filter to creates an image using the Perlin turbulence function
  • IE_ET_DistantLight: Filter to define a distant light source that can be used within a lighting filter primitive
  • IE_ET_PointLight: Filter to define a light source which allows to create a point light effect
  • IE_ET_SpotLight: Filter to define a light source which allows to create a spotlight effect
