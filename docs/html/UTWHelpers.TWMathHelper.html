class function IntMod(const num, by: Integer): Integer; static;

Perform a real mathematical modulo, that works also with negative numbers
NOTE This function should be used instead of the Delphi mod operand whenever the expected result should be equals to the mathematical modulo result. For example, processing -6 mod 5 is expected to return 4, conformly to a mathematical modulo result, however the Delphi modulo will return -1 instead

Parameters
num
Number on which modulo should be performed
by
Number against which modulo should be performed

class function Normalize(number, limit: Integer; out loops: Integer): Integer; inline; static;

Normalize number between 0 and limit, number of times its is contained in the limit (number of under or overflow), e.g.

  • Normalize(25, 24, loops) will return 1 and loops will be 1 (one overflow)

  • Normalize(-1, 24, loops) will return 23 and loops will be -1 (one underflow)

  • Normalize(60, 24, loops) will return 12 and loops will be 2 (two overflows)

Parameters
number
Number to normalize
limit
Max allowed number value (exclusive, e.g. if limit 24 and number 24 will return 0)
loops
[out] Fractional loops due to underflow/overflow
Returns

Normalized number
