Declaration

type TWSVGElement = class(TWSVGItem)

Description

A Scalable Vector Graphics (SVG) element is a data describing a part of the drawing or an action to be undertaken when drawing an SVG. It may contain a path to draw, a gradient to apply to a painting surface, an animation to apply to a shape, ...
