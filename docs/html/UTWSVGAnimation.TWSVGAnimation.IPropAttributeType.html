Nested Types

IEAttributeType = (...);

Attribute type enumeration

Values
  • IE_AT_Auto: The implementation should match the attributeName to an attribute for the target element. User-agents first search through the list of CSS properties for a matching property name, and if none is found, search the default XML namespace for the element
  • IE_AT_CSS: This specifies that the value of attributeName is the name of a CSS property defined as animatable
  • IE_AT_XML: This specifies that the value of attributeName is the name of an XML attribute defined as animatable in the default XML namespace for the target element
