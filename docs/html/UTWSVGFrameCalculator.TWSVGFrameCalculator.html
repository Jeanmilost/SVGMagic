Instead, the FPS represents the number of frames per seconds a system can effectively process

procedure SetDuration(duration: Double; frameCount: NativeUInt); virtual;

Calculate frame count and time interval to respect a total animation duration
NOTE The duration is based on the fact that 100% of an animation cycle must be performed during this time. E.g. for an animation that should execute 100 frames during 6 seconds, the frame count will be equal to 17
