Internal Types

EWAnimationTimerMessages = (...);

Animation messages that can be sent to observers
NOTE These values begin on 0 to not interfere with other messages. The allowed range for a new animation timer message is between 0 and 99

Values
  • IE_AM_Animate = 0: Message notifying that a new animation frame should be generated
  • IE_AM_Destroying: Message notifying that the animation timer is being destroyed
