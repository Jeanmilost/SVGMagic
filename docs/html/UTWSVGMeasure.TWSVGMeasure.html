IEUnit = (...);

Unit enumeration

Values
  • IE_UN_Unknown: Value for which unit is unknown
  • IE_UN_None: Value without unit
  • IE_UN_CM: Value expressed in centimeters
  • IE_UN_FT: Value expressed in feet
  • IE_UN_IN: Value expressed in inches
  • IE_UN_M: Value expressed in meters
  • IE_UN_MM: Value expressed in millimeters
  • IE_UN_PC: Value expressed in picas
  • IE_UN_PT: Value expressed in points
  • IE_UN_PX: Value expressed in pixels
  • IE_UN_Percent: Value expressed in percent
