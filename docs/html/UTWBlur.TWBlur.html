function Apply(pBitmap: Vcl.Graphics.TBitmap; const blurSize: TSize; resizeMode: IEResizeMode; gdiPlusToken: ULONG_PTR): Vcl.Graphics.TBitmap; overload; virtual;

Apply blur effect to bitmap by reducing the image then enlarging again
NOTE A frame corresponding to blur size values must be reserved around image, otherwise the resulting blured image will be cropped
NOTE Resulting bitmap must be deleted by caller
NOTE This function uses GDI+ internally. For this reason, GDI+ should be initialized externally and his token should be passed to this function

Parameters
pBitmap
Bitmap for which blur is applied
blurSize
Blur size
resizeMode
Resize mode (it's the interpolation algorithm used by GDI+)
gdiPlusToken
GDI+ token
Returns

Blured bitmap, Nil on error
