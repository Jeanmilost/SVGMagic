type TWGDIHelper = class(TObject)

Description

Optimized functions to draw with Windows GDI. These functions support alpha transparency
NOTE The code has been optimized to gain maximum possible speed, therefore, do not clean it unless you are really aware of what you are doing

Internal Types

IEBitmapKey = (...);

Cached bitmap keys enumeration

Values
  • IE_BK_DrawRectMaskDIB: Cached device independent bitmap used to draw rectangle mask
  • IE_BK_DrawRectOverlayDIB: Cached device independent bitmap used as overlay to draw rectangle
  • IE_BK_DrawTextMaskDIB: Cached device independent bitmap used to draw text mask
  • IE_BK_DrawTextOverlayDIB: Cached device independent bitmap used as overlay to draw text
IEDrawTextFunc = (...);

Draw text function to use

Values
  • IE_DF_GDI: Native GDI draw text function is used
  • IE_DF_WS_Optimized: Advanced draw text function is used
IEHAlign = (...);

Horizontal alignment

Values
  • IE_HA_None: Not aligned
  • IE_HA_Left: Left aligned
  • IE_HA_Center: Centered
  • IE_HA_Right: Right aligned
IETextTrimming = (...);

Text trimming

Values
  • IE_TT_None: Text is not trimmed
  • IE_TT_Character: For compatibility with GDI+, should not be used here (same result as IE_TT_None)
  • IE_TT_Word: For compatibility with GDI+, should not be used here (same result as IE_TT_None)
  • IE_TT_EllipsisCharacter: Text is ellipsified on the closest char
  • IE_TT_EllipsisWord: Text is ellipsified on the closest word
  • IE_TT_EllipsisPath: Text is ellipsified on the middle, string begin and end are visible
IEVAlign = (...);

Vertical alignment

Values
  • IE_VA_None: Not aligned
  • IE_VA_Top: Top aligned
  • IE_VA_Center: Centered
  • IE_VA_Bottom: Bottom aligned
