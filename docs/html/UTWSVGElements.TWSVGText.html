Nested Types

IEAnchor = (...);

Text anchor enumeration

Values
  • IE_TA_Start: Text is anchored on the left or top
  • IE_TA_Middle: Text is anchored on the center
  • IE_TA_End: Text is anchored on the right or bottom
IEDecoration = (...);

Text decoration enumeration

Values
  • IE_D_Normal: Text without decoration
  • IE_D_Underline: Underlined text
  • IE_D_LineThrough: Line through text
IEFontStyle = (...);

Text font style enumeration

Values
  • IE_FS_Normal: Normal font style
  • IE_FS_Italic: Italic font style
  • IE_FS_Oblique: Oblique font style, same as italic but with an optional angle value
