IEEdge = (...);

Hit edge enumeration

Values
  • IE_Top: Indicates that top edge was hit
  • IE_Left: Indicates that left edge was hit
  • IE_Bottom: Indicates that bottom edge was hit
  • IE_Right: Indicates that right edge was hit
  • IE_Center: Indicates that center was hit
IEEdges = set of IEEdge;
 
IEInvalidSize = (...);

Invalid size result enumeration

Values
  • IE_IS_None: Indicates that rect contains no invalid size
  • IE_IS_Width: Indicates that rect width is invalid
  • IE_IS_Height: Indicates that rect height is invalid
  • IE_IS_Both: Indicates that both rect width and height are invalid
