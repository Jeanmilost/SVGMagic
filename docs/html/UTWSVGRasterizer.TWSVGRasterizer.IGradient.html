Declaration

type IGradient = class(IPropItem)

Description

Gradient
NOTE The gradient unit also determines if the value should be read in percent or as coordinates. If IE_GU_ObjectBoundingBox is declared, then the values should be read in percent (between 0.0f and 1.0f). With IE_GU_UserSpaceOnUse, the values contain real coordinates, in the same units as the viewport
