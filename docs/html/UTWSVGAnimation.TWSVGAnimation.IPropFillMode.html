Internal Types

IEMode = (...);

Fill mode enumeration

Values
  • IE_FM_Unknown: The animation effect is unknown
  • IE_FM_Freeze: The animation effect is defined to freeze the effect value at the last value of the active duration. The animation effect is "frozen" for the remainder of the document duration (or until the animation is restarted)
  • IE_FM_Remove: The animation effect is removed (no longer applied) when the active duration of the animation is over. After the active end of the animation, the animation no longer affects the target (unless the animation is restarted)
