class function AddTo(const pGraphics: TWSVGGraphics; pImageList: TImageList): Boolean; static;

Add the content of a SVG graphic list in an image list
NOTE To support the alpha transparency, the image list should be configured as follow:

  • ColorDepth should be set to 32bit

  • DrawingStyle should be set to dsTransparent

  • Masked should be set to False


NOTE The SVG images will be added immediately after the last image in the list. The previous list content WILL NOT be deleted

Parameters
pGraphics
SVG graphic list to use to populate the image list
pImageList
Image list to populate
