function FromSMIL(const txt: UnicodeString): Boolean; virtual;

Set time from SMIL standard formatted string (used e.g. in SVG files)
NOTE SMIL (Synchronized Multimedia Integration Language) is a standard used to describe multimedia presentations. It defines markup for timing, layout, animations, visual transitions, and media embedding, among other things. SMIL is used by the SVG standard, to animate one or many shape contained inside. The SMIL time notation can be written as follow:

  • 02:30:03 = 2 hours, 30 minutes and 3 seconds

  • 50:00:10.25 = 50 hours, 10 seconds and 250 milliseconds

  • 02:33 = 2 minutes and 33 seconds

  • 00:10.5 = 10.5 seconds = 10 seconds and 500 milliseconds

  • 3.2h = 3.2 hours = 3 hours and 12 minutes

  • 45min = 45 minutes

  • 30s = 30 second

  • 5ms = 5 milliseconds

  • 12.467 = 12 seconds and 467 milliseconds

Fractional values are just (base 10) floating point definitions of seconds. Thus:

  • 00.5s = 500 milliseconds

  • 00:00.005 = 5 milliseconds

txt
SMIL time string
Returns

True on success, otherwise False

function NormalizeAndSet(hour, min, sec: Integer; msec: Integer = 0): Integer; virtual;

Set time and fix it for over flows, returns overflow in days. Values can be nagtives, e.g.

  • 23:13:60 becomes 23:14:00

  • 25:60:121 becomes 02:02:01 and will return 1 (one extra day)
