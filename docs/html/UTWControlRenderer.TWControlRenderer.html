class function GetFontSizeToFit(const text: UnicodeString; const rect: TWRectF; const pOptions: TWRenderer.ITextOptions; minWrapFontSize, minFontSize: Cardinal; hDC: THandle; out fontSize: Cardinal): Boolean; static;

Get font size so text can fit in rectangle
NOTE This function makes certain operations on the options.m_pFont pointer directly. By doing this, the original source font object is modified, even if restored after function ends. This may affect source object, such as e.g. calling his OnFontChange event

Parameters

Text to measure
rect

pOptions
Text GDI+ options
minWrapFontSize
Min font size after which text wrap is authorized
minFontSize
Minimum allowed font size
hDC
Device context handle
fontSize
[out] Result font size
Returns

True if font size is found for text to fit rect, False if still too big or on invalid argument
