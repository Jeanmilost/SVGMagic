IEImageAspectRatio = (...);

Image aspect ratio

Values
  • IE_AR_None: Do not force uniform scaling. Scale the graphic content of the given element non-uniformly if necessary such that the element's bounding box exactly matches the viewport rectangle. Note that if <align> is none, then the optional <meetOrSlice> value is ignored
  • IE_AR_XMinYMin: Force uniform scaling. Align the <min-x> of the element's viewBox with the smallest X value of the viewport. Align the <min-y> of the element's viewBox with the smallest Y value of the viewport
  • IE_AR_XMidYMin: Force uniform scaling. Align the midpoint X value of the element's viewBox with the midpoint X value of the viewport. Align the <min-y> of the element's viewBox with the smallest Y value of the viewport
  • IE_AR_XMaxYMin: Force uniform scaling. Align the <min-x>+<width> of the element's viewBox with the maximum X value of the viewport. Align the <min-y> of the element's viewBox with the smallest Y value of the viewport
  • IE_AR_XMinYMid: Force uniform scaling. Align the <min-x> of the element's viewBox with the smallest X value of the viewport. Align the midpoint Y value of the element's viewBox with the midpoint Y value of the viewport
  • IE_AR_XMidYMid: Force uniform scaling. Align the midpoint X value of the element's viewBox with the midpoint X value of the viewport. Align the midpoint Y value of the element's viewBox with the midpoint Y value of the viewport
  • IE_AR_XMaxYMid: Force uniform scaling. Align the <min-x>+<width> of the element's viewBox with the maximum X value of the viewport. Align the midpoint Y value of the element's viewBox with the midpoint Y value of the viewport
  • IE_AR_XMinYMax: Force uniform scaling. Align the <min-x> of the element's viewBox with the smallest X value of the viewport. Align the <min-y>+<height> of the element's viewBox with the maximum Y value of the viewport
  • IE_AR_XMidYMax: Force uniform scaling. Align the midpoint X value of the element's viewBox with the midpoint X value of the viewport. Align the <min-y>+<height> of the element's viewBox with the maximum Y value of the viewport
  • IE_AR_XMaxYMax: Force uniform scaling. Align the <min-x>+<width> of the element's viewBox with the maximum X value of the viewport. Align the <min-y>+<height> of the element's viewBox with the maximum Y value of the viewport
IEImageAspectRatioRef = (...);

Image aspect ratio reference

Values
  • IE_IR_Meet: Scale the graphic such that: - aspect ratio is preserved - the entire viewBox is visible within the viewport - the viewBox is scaled up as much as possible, while still meeting the other criteria In this case, if the aspect ratio of the graphic does not match the viewport, some of the viewport will extend beyond the bounds of the viewBox (i.e., the area into which the viewBox will draw will be smaller than the viewport)
  • IE_IR_Slice: Scale the graphic such that: - aspect ratio is preserved - the entire viewport is covered by the viewBox - the viewBox is scaled down as much as possible, while still meeting the other criteria
IEImageType = (...);

Image type

Values
  • IE_IT_Unknown: image type is unknown
  • IE_IT_PNG: The image is a PNG image
  • IE_IT_JPG: The image is a JPG image
  • IE_IT_SVG: The image is a SVG image
IEPropRule = (...);

Properties rules, determine how the prop item should be used in relation with his parent

Values
  • IE_PR_Default: The default value is used
  • IE_PR_Inherit: The value is inherited from the parent, set to default if no parent
  • IE_PR_Combine: The value is merged with the value contained in the parent
