Nested Types

IEAnimType = (...);

Animation type enumeration

Values
  • IE_AT_Unknown: Unknown animation target
  • IE_AT_Set: Specifies a set of values that modify a target attribute over time
  • IE_AT_Animate: Specifies an animation for a single target attribute or property over time
  • IE_AT_Animate_Color: Specifies a color transformation over time
  • IE_AT_Animate_Transform: Specifies an animation for a transformation attribute on a target element, thereby allowing animations to control translation, scaling, rotation and/or skewing
  • IE_AT_Animate_Motion: Specifies a target element to move along a motion path
