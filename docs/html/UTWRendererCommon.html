ETextRendering = (...);

Text rendering type enumeration
NOTE WARNING Don't change enum elements order or values, used in type conversions

Values
  • E_R_None: No text rendering type
  • E_R_Default: Each character is drawn using its glyph bitmap, with the system default rendering hint. The text will be drawn using whatever font-smoothing settings the user has selected for the system
  • E_R_AntiAlias: Each character is drawn using its antialiased glyph bitmap without hinting. Better quality due to antialiasing. Stem width differences may be noticeable because hinting is turned off
  • E_R_AntiAlias_GridFit: Each character is drawn using its antialiased glyph bitmap with hinting. Much better quality due to antialiasing, but at a higher performance cost
  • E_R_ClearType_GridFit: Each character is drawn using its glyph ClearType bitmap with hinting. The highest quality setting. Used to take advantage of ClearType font features
  • E_R_SingleBitPerPixel: Each character is drawn using its glyph bitmap. Hinting is not used
  • E_R_SingleBitPerPixel_GridFit: Each character is drawn using its glyph bitmap. Hinting is used to improve character appearance on stems and curvature
ETextTrimming = (...);

Text trimming type enumeration

Values
  • E_TT_None: No trimming
  • E_TT_Character: Trim on first char that exceeds the draw rect
  • E_TT_Word: Trim on first word that exceeds the draw rect
  • E_TT_EllipsisCharacter: Using ellipsis, trim on first char that exceeds the draw rect
  • E_TT_EllipsisWord: Using ellipsis, trim on first word that exceeds the draw rect
  • E_TT_EllipsisPath: Using ellipsis, cut the string on the middle and ellipsify the center
