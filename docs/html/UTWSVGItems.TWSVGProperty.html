Unit

UTWSVGItems

Declaration

type TWSVGProperty = class(TWSVGItem)

Description

A Scalable Vector Graphics (SVG) property is a data contained in an element, indicating the value of a particular variable. For example, a fill color, a stroke width, ...
NOTE In Inkscape files, properties containing the namespace sodipodi can be ignored, they are reminiscent of the first Inkscape application version, when it was still called SodiPodi
