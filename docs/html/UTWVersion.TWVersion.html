NOTE Format can be xx, xx.yy, xx.yy.zz or xx.yy.zz.ww, values are automatically swapped, i.e:

  • a string containing '8.0.3.0' will be converted to 8.0.3.0

  • a string containing '9.1.5' will be converted to 9.1.5.0

  • a string containing '8.25' will be converted to 8.25.0.0
