IEDataType = (...);

Data type

Values
  • IE_DT_URL: The value contains a classic URL or link
  • IE_DT_PNG: The value contains a PNG image data
  • IE_DT_JPG: The value contains a JPG image data
  • IE_DT_SVG: The value contains a SVG image data
  • IE_DT_Unknown: The value is unknown
