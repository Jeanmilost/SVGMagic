Description

Nested Types

IERestartType = (...);

Restart type enumeration

Values
  • IE_RT_Unknown: The animation restart behavior is unknown
  • IE_RT_Always: The animation can be restarted at any time. This is the default value
  • IE_RT_WhenNotActive: The animation can only be restarted when it is not active (i.e. after the active end). Attempts to restart the animation during its active duration are ignored
  • IE_RT_Never: The element cannot be restarted for the remainder of the current simple duration of the parent time container. (In the case of SVG, since the parent time container is the SVG document fragment, then the animation cannot be restarted for the remainder of the document duration)
