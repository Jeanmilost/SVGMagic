Declaration

type TWMatrix3x3 = record

Description

3x3 matrix
NOTE The matrix table can be read in both mathematical or in-memory order. From a mathematical point of view, the m_Table content should be read as m_Table[x][y], whereas it should be read as m_Table[y][x] from the in-memory point of view. The both representation are equivalent and interchangeable, and may be used anytime if the same logical is used constantly. However, this may be highly confusing when you have to manipulate the source code, because the table items may appear to be crossed. For that reason, be aware of what you are doing if you have to modify it
